Photoelectric Effect — Help

Physics

When light hits a metal cathode, electrons are ejected if photon energy exceeds the work function φ:

KE_max = hf − φ     (Einstein, 1905)
Vs = KE_max / e     (stopping potential)

The saturation current I₀ is proportional to photon density only. The stopping potential depends only on frequency, never on intensity.

I–V Curve

  • V ≥ 0: saturation current I₀ (flat)
  • −Vs ≤ V < 0: curve matching electron velocity distribution
  • V < −Vs: current = 0

Resources — Photoelectric Effect

Concept

The photoelectric effect is the emission of electrons when light falls on a metal surface. It was explained by Einstein in 1905 using the quantum hypothesis, earning him the Nobel Prize in 1921.

Key Observations

  • A threshold frequency f₀ exists — below it, no emission regardless of intensity
  • Emission is instantaneous — no time delay even at low intensity
  • Intensity affects current only — more photons = more electrons, not faster ones
  • Frequency determines KE — higher f = higher kinetic energy of emitted electrons

Equations

KEmax = hf − φ
eVs = KEmax → Vs = KEmax (in eV units)
f₀ = φ / h    (threshold frequency)
λ₀ = c / f₀    (threshold wavelength)

Guided Experiment

1
Select a metal from the Cathode Metal row. Note its work function φ.
2
Set Photon Density to 0.50. Set Voltage to 0 V.
3
Decrease wavelength (drag the spectrum bar left) until the readout shows ● EMITTING. This λ is near λ₀.
4
Now vary Photon Density — observe current changes, KE stays constant.
5
Set negative voltage to find stopping potential: the voltage at which current drops to zero.
6
Use Auto Sweep to record 10 data points across wavelengths automatically.
7
Open Show Graphs → KE vs Frequency. Verify the linear relationship.

Inference from Graph

  • Slope of KE vs f line = Planck's constant h
  • x-intercept = threshold frequency f₀
  • y-intercept = −φ (negative of work function)

Pedagogical Note: Negative KE values shown on the extrapolated graph line are theoretical. In physical reality, electrons cannot have negative kinetic energy; emission simply does not occur below the threshold frequency.
